# 📝 Quiz Management App

A fully functional Quiz Management Web Application built using **HTML, CSS, and JavaScript**.

This project allows users to:
- Register and Login
- Start Quiz
- Retake Quiz
- Create New Questions
- Delete Questions
- View Score Card
- Store Data using LocalStorage

---

# 🚀 Features

✅ User Registration System  
✅ User Login System  
✅ Home Page Navigation  
✅ Start Quiz Feature  
✅ Random Question Shuffle  
✅ Retake Quiz Option  
✅ Create Quiz Questions  
✅ Delete Quiz Questions  
✅ Score Card System  
✅ LocalStorage Data Saving  
✅ Responsive UI Design  
✅ Gradient Background UI
